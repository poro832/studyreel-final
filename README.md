# StudyReel — 기말 발표 덱

유튜브 쇼츠를 교육용 세로 피드로 바꾼 학습 앱 **StudyReel**의 기말 발표 자료(reveal.js).

- **발표 덱**: https://poro832.github.io/studyreel-final/
- **진행 현황(WBS)**: https://poro832.github.io/studyreel-final/wbs.html
- 앱 저장소: https://github.com/poro832/StudyReel

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 기말 5분 발표 덱 (14슬라이드) |
| `wbs.html` | WBS · 일정 페이지 |
| `assets/` | 화면·아키텍처 다이어그램·구동 GIF |
| `.nojekyll` | 정적 사이트 — Jekyll 빌드 비활성 |

## 발표 방법

`index.html`을 브라우저에서 열고 전체화면(F11), `→`/스페이스로 슬라이드 전환.
